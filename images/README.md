# Placeholder pour l'image de profil

Pour ajouter votre photo de profil :

1. Préparez une photo carrée (recommandé : 500x500px)
2. Nommez-la `profile.jpg`
3. Placez-la dans le dossier `images/`

## Outils pour préparer votre photo :

- **Crop/Resize** : https://www.iloveimg.com/crop-image
- **Remove Background** : https://www.remove.bg/
- **Compress** : https://tinypng.com/

## Alternative : Utiliser une photo en ligne

Dans `index.html`, ligne 102, vous pouvez également utiliser une URL :

```html
<img src="https://votre-url-image.com/photo.jpg" alt="Photo de profil">
```

## Services de photos de profil :

- **Gravatar** : https://gravatar.com/
- **Unsplash** : https://unsplash.com/ (photos gratuites)
- **UI Faces** : https://uifaces.co/ (avatars)
